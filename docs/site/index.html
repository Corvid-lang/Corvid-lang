Corvid Roadmap Spec Inventions

General-purpose. AI-native. Compiler-enforced.

The language where AI behavior is part of the program.

Corvid is not a RAG framework and not another SDK wrapper. It is a general programming language where agents, tools, prompts, approvals, grounding, budgets, confidence, routing, streaming, replay, and verification are visible to the compiler.

Read the catalog
refund.cor 
effect transfer_money:
    cost: $0.50
    trust: human_required
    reversible: false

tool refund(id: String) -> Receipt dangerous uses transfer_money

@budget($1.00)
@trust(human_required)
agent safe_refund(id: String) -> Receipt:
    approve Refund(id)
    return refund(id)

Claim discipline

No shortcut marketing.

This page does not say Corvid is faster than Python, safer than TypeScript, or formally verified unless the claim links to a command that runs. Corvid's advantage is not a slogan; it is the set of language features above, each tied to a source file, spec, and test.
